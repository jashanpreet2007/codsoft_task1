✅ CODSOFT Task 1 - To-Do List Application

📌 Project Overview

This project is a desktop-based To-Do List Application developed using Python and Tkinter as part of the CODSOFT Python Programming Internship.

The application provides a simple and intuitive interface for users to organize daily tasks efficiently. Users can add new tasks, mark them as completed, delete selected tasks, and clear the entire task list.

---

✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete selected tasks
- 🧹 Clear all tasks with confirmation
- ⚠️ Input validation and warning messages
- 🖥️ Simple and user-friendly graphical interface

---

🛠️ Technologies Used

- Python 3
- Tkinter
- MessageBox
